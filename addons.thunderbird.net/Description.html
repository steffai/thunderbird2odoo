Odoo Email Connector is a Thunderbird add-on that allows you to export emails from Thunderbird into the ERP software Odoo. An email status card appears above each message, showing whether it is already known in Odoo with direct links to the corresponding record. Requirements
  • Odoo version 19 or newer
  • Optional Odoo extension:
    • mail_manual_routing improves handling of new or unrelated emails that might otherwise not be visible in Odoo. Without this extension, emails imported as Generic may not appear. Imported as Opportunity (CRM Lead) they work in standard Odoo installations.
Setup
  • Odoo:
    • Create an API key: User → My Preferences → Security → API Keys → Add API Key
  • Thunderbird
    • Install the add-on Odoo Email Connector
    • Open the add-on settings and configure:
      • Odoo server URL
      • API key
      • Odoo database name (usually not required)
    • Test the connection. You will be prompted to grant host permission to access your Odoo server, then save the settings
Usage
  • Email status card — When viewing an email, a status bar at the top shows whether it is known in Odoo (found with a green dot, not found with a red cross). Click the Odoo links to open the record. If only a predecessor email is known, the bar shows an amber dot and a link to the parent.
  • Verify — Click the Verify button in the status card to check if the current email exists in Odoo. You can also verify multiple emails at once via right-click → Odoo Email Connector → Verify in the message list.
  • Import — Right-click an email and select Odoo Email Connector → Import this email. If a predecessor (In-Reply-To / References) is already in Odoo, you will be asked to confirm before importing.
  • Sync from Odoo — Right-click in the message list and select Odoo Email Connector → Sync from Odoo to batch-cache the Odoo status of recent emails, making the status card show results instantly without individual lookups.
