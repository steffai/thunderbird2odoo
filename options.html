Odoo Email Connector

Links Thunderbird emails to Odoo objects — verify, import, and sync message IDs with your Odoo instance.

How it works
  • Status bar — when viewing an email, shows whether it is known in Odoo (cache only, no server request). Only appears if the email has cached data; otherwise hidden.
  • Verify — checks if the email exists on the Odoo server
  • Add / Import — uploads the email to Odoo
  • Sync from Odoo — batch-caches message IDs for instant status display
Data exchanged with your Odoo server: No data is sent automatically — only when you click a button.
  • Verify: queries the server with the email's Message-ID, In-Reply-To ID and Reference IDs headers
  • Add / Import: uploads the complete email including headers, body, and attachments
  • Sync: downloads email metadata (Message-ID, Odoo model name, record ID, Odoo message ID) — no email content transferred
